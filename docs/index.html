What is OpenPolygon?

OpenPolygon is a simple Rendering Engine for Developer to creating objectives , effects with Shader (GLSL) Codes. OpenPolygon include a lightweight entity-component-system to easy create any objectives. Otherwise include is a lightweight Render Pipeline to create and manipulate FrameBuffers.

Next Project?

The next Project is a Polygon Framework. A Collection of objectives, effects and other Engines like Physic, Sound-Engines.
